DevOps Lifecycle - Gaming Website Development

Plan

Defined project goals: immersive video backgrounds, product showcases, animations, and responsive layout.

Develop

Wrote HTML, CSS, and JS; integrated Cloudinary-hosted assets; added interactive sections and animations.

Build

Optimized media, minified code, and prepared deployment-ready files.

Test

Tested across devices/browsers, checked animations, video performance, and responsive design.

Release

Approved the final version and prepared server/domain settings for launch.

Deploy

Uploaded files to GoDaddy/InfinityFree hosting and verified all live links & videos.

Operate

Ensured smooth live performance and monitored uptime and user experience.

Monitor

Analyzed analytics, load times, and user feedback for continuous improvement.
